CarGuarantee is a fully responsive and user-friendly auto maintenance & repair service website.
It allows customers to explore offered services, request quotes, and track their repair progress —
with an Admin Dashboard for managing service requests.

| Page             | Description                                         |
| ---------------- | --------------------------------------------------- |
| `index.html`     | Main homepage introducing CarGuarantee services     |
| `quote.html`     | Quote request form for customers                    |
| `track.html`     | Track page for checking service status              |
| `admin.html`     | Admin login page                                    |
| `dashboard.html` | Admin dashboard to view and update service requests |

Features
 Frontend

Modern, clean, responsive UI (works across all devices)

Navigation links for Home, Services, Track, Contact, Admin

Hero banner showcasing business highlights

Services section with images and detailed info

Work/projects showcase with clickable previews

Contact information and social media integration

“Get a Quote” button linking to the quote form

 Admin System

Secure login form (admin.html) with credential validation
(Default username: admin, password: admin123)

Dashboard (dashboard.html) to view customer requests

Interactive action buttons to:

Mark service as completed

View customer details

Alerts and live status updates on each action

Logout feature redirects back to the admin login page
